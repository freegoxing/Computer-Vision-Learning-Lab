# YOLOv8 Learning / Tutorial

## Overview
This repository contains my personal learning and experiments with YOLOv8 for object detection. 
It includes tutorials, dataset preparation, training, evaluation, and model deployment examples.

## Contents
- `src/` : Training, evaluation, and inference scripts.
- `models/` : Trained weights and checkpoints (optional, can be in GitHub releases).
- `runs/` : Example outputs and visualizations.

## Requirements
- Python >= 3.12
- torch, torchvision
- ultralytics (YOLOv8)
- OpenCV, matplotlib

Install dependencies:
```bash
uv sync
```
